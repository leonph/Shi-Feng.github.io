ElastoPlasticBiotFoam求解器解析

Posted on 2017-12-27 | In OpenFOAM | Visitors | times

基本介绍

ElastoPlasticBiotFoam求解器是TANG TIAN完成其博士论文研究过程中编写的求解器，主要采用有限体积法进行饱和土的比奥固结问题分析。 其中土体本构采用理想塑性摩尔库伦本构模型。
求解器的详细理论说明可参考TANG TIAN的论文：Tang, T., Hededal, O. and Cardiff, P., 2015. On finite volume method implementation of poro‐elasto‐plasticity soil model. International journal for numerical and analytical methods in geomechanics, 39(13), pp.1410-1430.

代码分析

  • 求解器主程序
for (runTime++; !runTime.end(); runTime++)
{
    Info<< "Time: " << runTime.timeName() << nl << endl;
    #include "readElastoPlasticBiotFoamControls.H" //读取求解器的控制参数
    int iCorr = 0; //迭代次数
    scalar initialResidual = GREAT;  //迭代初始残差     
    lduMatrix::solverPerformance solverPerfP;
    lduMatrix::solverPerformance solverPerfDU;

    // gives an option for switching off the plasticity
    // solves the original poro-elastic Biot's equations
    word correct=word(runTime.controlDict().lookup("correctPlasticity")); //是否修正塑性

    do 
      {
        iCorr++;

        // store previous iteration, since later p will be under-relaxed    
        p.storePrevIter(); //后期压力场放松使用

        // solve the pore pressure equation 求解压力方程
        fvScalarMatrix pEqn             
        (
                  fvm::ddt(p) == fvm::laplacian(Dp, p) - fvc::div(fvc::ddt(Dp2,U))             
        );
        solverPerfP = pEqn.solve(); 
        p.relax();//压力场放松
        dp = p - p.oldTime();//更新孔压增量
        dp.correctBoundaryConditions();//因孔压属于方程求解量，更新孔压增量边界值

        // store previous iteration, since later dU will be under-relaxed        
        dU.storePrevIter();//后期位移增量场放松使用

        // calculate the explicit part of div stress 
        #   include "calculateDivDsigmaExp.H"//显示计算应力相关项

        // solve the momentum (soil skeleton displacement) equation with 
        // 'lagged' pressure coupling, plasticity and cross-component coupling
        fvVectorMatrix dUEqn
        (
            fvm::laplacian(2.0*mu + lambda, dU, "laplacian(dU)")
         ==
            - divDsigmaExp 
            + fvc::div(2.0*mu*(mesh.Sf() & fvc::interpolate(dEpsP)))
            + fvc::div(lambda*(mesh.Sf() & I*fvc::interpolate(tr(dEpsP))))
            + fvc::grad(dp) 
        ); 
        solverPerfDU = dUEqn.solve();

        dU.relax(); //位移场放松

        gradDU = fvc::grad(dU);//更新位移梯度

        // update stress and plastic strains if plasticity option is switched on           
        if (correct=="yes")
        {
          # include "correctPlasticity.H" //修正塑性，此处采用的是摩尔库伦模型
        } 

        initialResidual = max(solverPerfP.initialResidual(),solverPerfDU.initialResidual());

        U = U.oldTime()+dU;
        U.correctBoundaryConditions();

       if (iCorr % 10 == 1){ Info << "\tTime " << runTime.value()
         << ", Corrector " << iCorr
         << ", Solving for " << dU.name()
         << ", residual = " << initialResidual << endl;   
          }          

      } while (initialResidual > convergenceTolerance && iCorr < nCorr); //收敛精度判断，满足，下一时间步，不满足，下一迭代步

其中主要涉及两个方程的求解，一是孔压方程的求解，二是位移增量方程的求解。孔压方程的求解部分相对简单易懂，位移增量方程的求解部分相对难懂，
需要结合论文熟悉其理论知识，包括为优化收敛性所做的左右laplacian项平衡处理，以及塑性修正部分的理论（难点在于找到合适的应力积分算法）。
